## Sobre o Projeto

**Demeter** é um app iOS interativo que combina **astrologia e botânica**: a partir do seu signo, você descobre qual planta está ligada a você, aprende sobre o que ela representa espiritualmente, como ela se relaciona com seu signo e as melhores formas de cultivo.

> Aprenda sobre plantas de forma interativa, divertida e personalizada!

## Tecnologias

- **Swift** — linguagem principal do desenvolvimento
- **UIKit** — componentes nativos iOS
- **AutoLayout** — layouts responsivos para múltiplos dispositivos
- **Figma** — protótipo e assets criados no Figma
- **Xcode** — ambiente de desenvolvimento

---

## Como Executar

1. Clone ou baixe este repositório
2. Abra o arquivo `Demeter.xcodeproj` no **Xcode**
3. Selecione um simulador ou dispositivo físico
4. Pressione **Cmd + R** para rodar
